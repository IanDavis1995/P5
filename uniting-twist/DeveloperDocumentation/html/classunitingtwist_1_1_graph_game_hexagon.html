Detailed Description

Represents a hexagon space in our FieldGraph.

This contains a GameHexagon that can be an empty cell when a hexagon has not yet landed there or is cleared away. Created by Litun on 12.04.2016.

int compareTo ( @NonNull GraphGameHexagon  other)

Comparison override for GraphGameHexagons.

The comparison behavior is as follows: We are greater than other if our row or column in the graph is greater than theirs. We are less than other if our row or column in the graph is less than theirs. We are equal if we have the same row and column in the graph.

Parameters
otherThe other GraphGameHexagon to compare to.
Returns
1 if this > other -1 if this < other 0 if this == other
