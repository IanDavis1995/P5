Detailed Description

Utility class with a library of basic color definitions for OpenGL shading.

This class creates the following predefined colors: 

 greyColor (Grey)

 redColor (Red)

 yellowColor (Yellow)

 blueColor (Blue)

 oceanColor (Ocean)

 greenColor (Green)

They are accessed using the ColorUtils.getColor function.
