\subsection{Detailed Description}
Utility class with a library of basic color definitions for Open\+GL shading. 

This class creates the following predefined colors\+: \begin{DoxyVerb} greyColor (Grey)

 redColor (Red)

 yellowColor (Yellow)

 blueColor (Blue)

 oceanColor (Ocean)

 greenColor (Green)
\end{DoxyVerb}


They are accessed using the \doxyref{Color\+Utils.\+get\+Color}{p.}{classunitingtwist_1_1_color_utils_aa3a7d7ee785a482b0ff389374266933a} function.
